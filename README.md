# Portfolio Hub

Minimal terminal-themed portfolio hub website for GitHub Pages.

**Live site:** https://jellybeansontoasties.github.io

## Design

Terminal aesthetic inspired by kitty terminal config with orange accents (`#FF8C00`) on dark earth background (`#1B0906`).

## Tech

- Pure HTML/CSS
- JetBrains Mono typography
- No JavaScript
- GitHub Pages compatible

## Structure

```
Jellybeansontoasties.github.io/
├── index.html              # Landing page
├── assets/
│   └── css/
│       └── style.css      # Design system
├── .nojekyll              # Disable Jekyll
└── README.md              # This file
```
